# Password Strength Auditor & Breach Checker

This project is a Python-based cybersecurity tool designed to evaluate password strength and identify common weaknesses, including known breached passwords.

---

## Features
- Password strength analysis
- Detection of weak patterns and dictionary words
- Secure SHA-256 password hashing
- Real-time breach password checking via HIBP API
- JSON security report generation

---

## Usage

```bash
python3 auditor.py
